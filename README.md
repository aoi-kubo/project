# my-project
都道府県別の総人口推移グラフを表示するSPA(Single Page Application)の作成。  
RESAS-API(地域経済分析システム)から人口データを受け取り、Highchartsを用いてグラフを描画。

HTML Scss Vue.js  
RESAS-API Highcharts
